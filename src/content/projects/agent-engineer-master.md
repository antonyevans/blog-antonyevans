## What it is

Agent Engineer Master (AEM) is a skill-as-a-service for Claude Code. You submit a brief describing what you want an AI agent to do. AEM delivers a production-ready skill folder — `SKILL.md`, reference documents, and eval files — that you drop directly into your Claude Code setup.

## Why it exists

The gap is not agent hosting or builder platforms — those exist. The gap is *skill generation*: a self-serve service that takes "here's what I need my agent to do" and returns a well-structured, tested, production-ready skill file. Freelance consultants charge $100–$300/hr for this. AEM productises it.

## What it does

- Takes a structured brief: task description, trigger logic, tools used, human-in-the-loop points, output contract
- Runs the full skill engineering process: prior art research, design, generation, preflight validation
- Delivers a complete skill folder ready to install via a single copy into `.claude/skills/`
- Free sample skills demonstrate the quality bar before purchase
